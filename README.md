# all-in-one-seo-pack

## Project setup
```
npm install
```

### Compiles and hot-reloads for development (Lite)
```
npm run dev:lite
```

### Compiles and hot-reloads for development (Pro)
```
npm run dev:pro
```

### Compiles and minifies for production (Lite)
```
npm run build:lite
```

### Compiles and minifies for production (Pro)
```
npm run build:pro
```
